🔷 HTML Elementi Nedir?

📖 Temel Yapı
<etiket> içerik </etiket>

Bir element genellikle:

  • ✅ Açılış etiketi: <p>
  • ✅ Kapanış etiketi: </p>
  • ✅ İçerik: Açılış ve kapanış arasındaki şey
  • ✅ Kendi kendini kapatabilir: <img> (kapanış etiketi yok)
  • ✅ Öznitelik (attribute) alabilir: src="resim.jpg"
  • ✅ İç içe elementler barındırabilir: <p> <em>vurgu</em> </p>

🔷 Elementler İç İçe Olabilir

<p>
  HTML is <em>awesome!!!</em>
  <img src="puppy.png" />
</p>

✨ Canlı Önizleme

HTML is awesome!!! (resim placeholder)

📝 Not: <em> elementi <p>'nin içinde. <img> ise kendi kendini kapatan bir element.

🎯 CSS ile Hizalama (Giriş Seviyesi)

📍 Metin Hizalama

En temel hizalama: text-align özelliği ile metinleri hizalayabilirsin.

text-align: left;
Bu metin solda hizalanır.
text-align: center;
Bu metin ortada hizalanır.
text-align: right;
Bu metin sağda hizalanır.
/* CSS kodu */
.text-left   { text-align: left; }
.text-center { text-align: center; }
.text-right  { text-align: right; }
📐 Flexbox ile Elementleri Hizalama (Giriş)

Flexbox, elementleri yan yana veya ortalamak için kullanılır. İlerleyen slaytlarda çok daha detaylı göreceğiz!

✨ Yan yana hizalama (justify-content: center)

🔹 Kutu 1 🔹 Kutu 2 🔹 Kutu 3

✨ İki yana yaslama (justify-content: space-between)

⬅️ Sol ➡️ Sağ 
/* Flexbox temel kullanımı */
.flex-center {
  display: flex;
  justify-content: center;  /* yatayda ortalar */
  gap: 1rem;                /* kutular arası boşluk */
}

.flex-between {
  display: flex;
  justify-content: space-between; /* iki uca yaslar */
}
🔮 İlerleyen slaytlarda: Flexbox, Grid, margin:auto, position, float gibi daha fazla hizalama yöntemi göreceğiz!

<p class="highlight" id="first-paragraph">Öznitelikler ek bilgi verir</p>
  • src → Resmin kaynak dosyasını belirtir
  • alt → Resim yüklenmezse gösterilen alternatif metin
  • href → Linkin gideceği adres
  • class / id → CSS veya JavaScript için tanımlayıcılar
  • target="_blank" → Linki yeni sekmede açar
