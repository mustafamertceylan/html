🤔 HTML/CSS Neden Bu Kadar Tuhaf?

Cevap: Öğrenmediğimiz çok önemli bir kural var — block vs inline display!

📖 HTML Nedir?

HTML (Hypertext Markup Language), bir web sayfasının içeriğini ve yapısını tanımlar. Yapı taşlarına element denir.

<p>
  HTML is <em>awesome!!!</em>
  <img src="puppy.png" />
</p>

HTML elementleri 3 temel tipe ayrılır:

  1. Block — büyük içerik blokları, yükseklik ve genişliği vardır
  2. Inline — küçük içerik miktarı, yükseklik/genişlik yoktur
  3. Inline-Block — inline gibi akar ama width/height alabilir

🔷 Block Elementler

📦 display: block;

Block elementler sayfanın tam genişliğini kaplar (yukarıdan aşağıya akar) ve yükseklik/genişlik özelliklerine sahiptir.

<p> <h1>...<h6> <blockquote> <ol> <ul> <li> <div> <section> 
<h1>About vrk</h1>
<p>
  She likes <em>puppies</em>
</p>

✅ Block Özet:
  • Tam genişlik kaplar (yukarıdan aşağıya akar)
  • Yükseklik ve genişlik özelleştirilebilir
  • İçinde block veya inline elementler barındırabilir

🔹 Inline Elementler

📝 display: inline;

Inline elementler sadece ihtiyaç duyduğu kadar genişlik kaplar (soldan sağa akar) ve yükseklik/genişlik özelliklerine sahip olamaz.

  • Soldan sağa akar (yan yana dizilir)
  • Sadece içerik kadar genişlik kaplar
  • width ve height özellikleri ÇALIŞMAZ
  • İçinde block element bulunduramaz

🔶 Inline-Block Elementler

⚡ display: inline-block;

Inline-Block elementler inline gibi soldan sağa akar (yan yana dizilir) AMA block gibi width ve height özelliklerine sahiptir!

<img> display: inline-block (CSS ile)
🎯 İnline-Block'a Width ve Height Verme: 
/* 1. YÖNTEM: img etiketi zaten inline-block'tur */
img {
  width: 50px;      /* ✅ ÇALIŞIR */
  height: 50px;     /* ✅ ÇALIŞIR */
}

/* 2. YÖNTEM: Herhangi bir elementi inline-block yapmak */
.kutum {
  display: inline-block;
  width: 150px;     /* ✅ ÇALIŞIR */
  height: 100px;    /* ✅ ÇALIŞIR */
  margin: 10px;     /* ✅ ÇALIŞIR */
  padding: 15px;    /* ✅ ÇALIŞIR */
}

📌 Inline-Block Özet:

  • ✅ İçinde block element bulundurabilir
  • 🎯 Menüler, kart galerileri, buton grupları için idealdir!

🔄 display CSS Özelliği ile Tip Değiştirme

Bir elementin varsayılan görüntüleme tipini display özelliği ile değiştirebilirsiniz!

/* Inline bir elementi block yapmak */
a {
  display: block;
}

/* Block bir elementi inline yapmak */
p {
  display: inline;
}

/* Herhangi bir elementi inline-block yapmak */
.my-element {
  display: inline-block;
}

📺 display ile Tip Değiştirme Örneği

🔹 Bu div aslında block ama display: inline yapıldı
🔹 İkinci div de inline oldu, yan yana dizildiler!

⬆️ Normalde <div> block elementtir, ama display: inline ile yan yana dizildi!

🎯 display için olası değerler:
  • display: block; → element block gibi davranır
  • display: inline; → element inline gibi davranır
  • display: inline-block; → element inline-block gibi davranır
  • display: none; → elementi tamamen gizler
  • display: flex; → flexbox container yapar
  • display: grid; → grid container yapar

📋 Genel Özet

  • 🔷 Block: Tam genişlik, yukarıdan aşağıya, width/height var → <div>, <p>, <h1>
  • 🔹 Inline: İçerik kadar genişlik, soldan sağa, width/height YOK → <span>, <a>, <strong>
  • 🔶 Inline-Block: Soldan sağa akar AMA width/height var → <img>, veya display: inline-block ile
  • 🔄 display özelliği ile bir elementin tipini değiştirebilirsin!
  • 🎯 Inline elemente width/height vermek istiyorsan → display: inline-block kullan!
