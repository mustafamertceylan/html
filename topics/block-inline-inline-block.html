📐 Block · Inline · Inline-Block

CSS display özelliği ile elementlerin davranışını kontrol etme

🔷 Block Elementler

📦 display: block;

Block elementler her zaman yeni bir satırda başlar ve bulunduğu kabın tüm genişliğini kaplar.

<div>Ben bir block elementim</div>
<p>Ben de bir block elementim</p>
<h1>Başlıklar da block elementtir</h1>

📌 Block Özellikleri:
  • ✅ Yeni satırda başlar
  • ✅ Tüm genişliği kaplar
  • ✅ width/height uygulanabilir
  • ✅ margin/padding her yönde çalışır

🔹 Inline Elementler

📝 display: inline;

Inline elementler yeni satır başlatmaz, yan yana dizilir ve sadece içeriği kadar yer kaplar.

<span>Ben inline elementim</span>
<a href="#">Link de inline&l;/a>
<strong>Kalın metin inline&l;/strong>

📺 Canlı Demo

🔸 Inline 1 🔸 Inline 2 🔸 Inline 3 🔸 Inline 4

⬆️ Tüm elementler yan yana dizildi!

📌 Inline Özellikleri:
  • ✅ Yan yana dizilir (yeni satır başlatmaz)
  • ✅ Sadece içerik kadar yer kaplar
  • ❌ width/height ÇALIŞMAZ
  • ⚠️ margin sadece sağ/sol çalışır

🔶 Inline-Block Elementler

⚡ display: inline-block;

Inline-Block elementler inline gibi yan yana dizilir ama block gibi width/height ve margin/padding özellikleri alır. İki dünyanın en iyisi!

.kutu {
  display: inline-block;
  width: 120px;
  height: 80px;
  margin: 10px;
  padding: 10px;
}

💡 Hangi Durumda Hangisini Kullanmalı?

📦 Block Kullan

Sayfa yapısı oluştururken, bölümleri ayırırken, içerik gruplarken → <div>, <section>, <header>

📝 Inline Kullan

Metin içinde küçük vurgular, linkler, stil değişiklikleri için → <span>, <a>, <strong>, <em>

⚡ Inline-Block Kullan

Menü butonları, kart galerileri, yatay form elemanları, satır içinde boyutlandırma gereken durumlar

/* Menü butonları için inline-block harikadır! */
.menu-item {
  display: inline-block;
  padding: 10px 20px;
  margin: 0 5px;
  background: #e8ff47;
  color: #000;
  border-radius: 8px;
}

/* Kart galerisi için */
.card {
  display: inline-block;
  width: 250px;
  margin: 10px;
  padding: 15px;
  border: 1px solid #ccc;
  border-radius: 8px;
}

✏️ Pratik Alıştırma

🎯 Kendin Dene!

Aşağıdaki CSS kodunu sayfana ekle ve sonuçları gözlemle:

.block-ornek {
  display: block;
  background: #3a3a4a;
  margin: 10px;
  padding: 10px;
}

.inline-ornek {
  display: inline;
  background: #3a3a4a;
  margin: 10px;    /* sadece sağ/sol çalışır! */
  padding: 10px;
}

.inline-block-ornek {
  display: inline-block;
  background: #3a3a4a;
  margin: 10px;
  padding: 10px;
  width: 150px;
}

📋 Özet

  • ✅ Block: Yeni satır, tam genişlik → Sayfa yapısı için
  • ✅ Inline: Yan yana, içerik kadar → Metin içi öğeler için
  • ✅ Inline-Block: Yan yana + width/height → Menü, galeri için
  • ✅ display özelliği ile bir elementin tipini değiştirebilirsin!
  • ✅ span'i display: block; ile block yapabilirsin
  • ✅ div'i display: inline; ile inline yapabilirsin
