📌 List Item (Liste Öğesi)

<li> etiketi, HTML'de listelerin her bir elemanını tanımlamak için kullanılır. Tek başına kullanılmaz, mutlaka bir liste kabı içinde olmalıdır.

<ul>
  <li>Elma</li>
  <li>Armut</li>
  <li>Muz</li>
</ul>

📺 Canlı Önizleme

  • Elma
  • Armut
  • Muz

🔷 Liste Türleri ve <li>

<li> etiketi iki farklı liste kapsayıcısı ile kullanılır:

📌 Sırasız Liste (<ul> + <li>)

<ul>
  <li>Kahve</li>
  <li>Çay</li>
  <li>Süt</li>
</ul>
  • Kahve
  • Çay
  • Süt

🔢 Sıralı Liste (<ol> + <li>)

<ol>
  <li>İlk adım</li>
  <li>İkinci adım</li>
  <li>Üçüncü adım</li>
</ol>
  1. İlk adım
  2. İkinci adım
  3. Üçüncü adım
📌 Fark: <ul> madde işaretli (•) liste oluştururken, <ol> numaralı (1,2,3) liste oluşturur.

🔷 İç İçe Listeler (Nested Lists)

<li> etiketinin içine başka bir liste koyarak alt listeler oluşturabilirsiniz.

🔷 <li> Öznitelikleri

📌 value (Sadece <ol> içinde)

Liste numaralarını belirli bir değerden başlatmak veya atlamak için kullanılır.

<ol>
  <li>Birinci</li>
  <li value="5">Beşinci (atlamalı)</li>
  <li>Altıncı</li>
</ol>
  1. Birinci
  2. Beşinci (atlamalı)
  3. Altıncı

📌 type (Eski yöntem, CSS tercih edilir)

Madde işareti veya numara tipini belirler.

<ol type="A">
  <li>A harfi ile</li>
  <li>Devam eder</li>
</ol>

<ul type="square">
  <li>Kare madde işareti</li>
</ul>
  1. A harfi ile
  2. Devam eder
  • Kare madde işareti
💡 Öneri: type özniteliği yerine CSS kullanmak daha modern ve esnektir:
list-style-type: upper-alpha; /* A, B, C */
list-style-type: square; /* Kare madde işareti */

🎨 CSS ile <li> Stillendirme

📌 Madde İşareti Tipini Değiştirme

/* Madde işareti tipi */
ul {
  list-style-type: square;  /* kare, circle, disc, none */
}

/* Sıralı liste numara tipi */
ol {
  list-style-type: upper-roman;  /* I, II, III */
}
  • Kare madde işareti
  • İkinci öğe
  1. Roma rakamı (I)
  2. Roma rakamı (II)

📌 Özel Stil: Madde İşaretlerini Kaldırma

.custom-list {
  list-style: none;
  padding-left: 0;
}

.custom-list li {
  background: #f0f0f0;
  margin: 5px 0;
  padding: 10px;
  border-radius: 5px;
}
  • ✨ Özel stilli liste öğesi 1
  • ✨ Özel stilli liste öğesi 2
  • ✨ Özel stilli liste öğesi 3

📌 Yatay Menü (li ile)

.menu {

📋 Özet

  • 📌 <li> = Liste öğesi (List Item)

  • 🔢 <ol> → Numaralı liste (1, 2, 3...)
  • • <ul> → Madde işaretli liste (•, -, *)
  • 📁 İç içe listeler oluşturulabilir
  • 🎨 CSS ile tamamen özelleştirilebilir (renk, arka plan, hizalama)
  • ⚠️ Tek başına kullanılamaz, mutlaka bir liste kabına ihtiyaç duyar
