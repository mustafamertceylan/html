CSS Box Model — Kutu Modeli (Detaylı)

CSS Box Model: Her HTML elementi bir kutudur. Content (içerik), Padding (iç boşluk), Border (kenarlık), Margin (dış boşluk) katmanlarının detaylı açıklaması. Toplam boyut hesaplama, box-sizing: border-box, margin collapse, pratik örnekler ve sık yapılan hatalar.

📦 CSS Box Model

Her HTML elementi bir kutudur — Margin · Border · Padding · Content

📖 Box Model Nedir?

CSS Box Model (Kutu Modeli), tüm HTML elementlerinin birer dikdörtgen kutu olarak ele alındığını açıklayan temel bir kavramdır. Her kutu 4 katmandan oluşur:

📐 MARGIN (Dış Boşluk)
🖌️ BORDER (Kenarlık)
📏 PADDING (İç Boşluk)
📄 CONTENT (İçerik)
Burada içerik var!
Metin, resim, diğer elementler...
🎯 Box Model'in 4 Katmanı (İçten Dışa):
  1. Content (İçerik) — Metin, resim veya diğer elementlerin bulunduğu alan
  2. Padding (İç Boşluk) — İçerik ile border arasındaki boşluk
  3. Border (Kenarlık) — Padding ve margin arasındaki çizgi/kenarlık
  4. Margin (Dış Boşluk) — Bu kutunun diğer kutularla arasındaki dış boşluk

📄 1. CONTENT — İçerik Alanı

Content, kutunun en iç kısmıdır. Metin, resim, video veya diğer HTML elementleri bu alanda bulunur.

/* Content özellikleri */
width: 300px;      /* İçerik genişliği */
height: 200px;     /* İçerik yüksekliği */
min-width: 100px;  /* Minimum genişlik */
max-width: 500px;  /* Maksimum genişlik */

📺 Content Örneği

width: 200px
height: 80px

⬆️ Bu kutunun content alanı 200x80 pikseldir.

💡 Önemli: width ve height varsayılan olarak sadece content alanını belirler. Padding ve border buna eklenir!

📏 2. PADDING — İç Boşluk

Padding, içerik ile border arasındaki iç boşluk alanıdır. Kutunun içindeki elementlerin kenarlara yapışmasını engeller.

/* Padding kullanımı */
padding: 10px;              /* 4 yönde de 10px */
padding: 10px 20px;         /* üst/alt:10px, sağ/sol:20px */
padding: 10px 20px 30px;    /* üst:10px, sağ/sol:20px, alt:30px */
padding: 10px 20px 30px 40px; /* üst, sağ, alt, sol (saat yönünde) */

/* Tek yönlü padding */
padding-top: 10px;
padding-right: 20px;
padding-bottom: 30px;
padding-left: 40px;

🖌️ 3. BORDER — Kenarlık

Border, padding ve margin arasındaki kenarlık çizgisidir. Stil, kalınlık ve renk olarak özelleştirilebilir.

/* Border kullanımı */
border: 2px solid black;     /* kalınlık, stil, renk */
border-width: 2px;
border-style: solid;         /* solid, dotted, dashed, double, groove, ridge, inset, outset, none */
border-color: #ff0000;

/* Tek yönlü border */
border-top: 2px solid red;
border-right: 2px dashed green;
border-bottom: 2px dotted blue;
border-left: 2px double yellow;

/* Border radius (yuvarlak köşeler) */
border-radius: 10px;         /* 4 köşe de 10px */
border-radius: 10px 20px;    /* sol-üst/sağ-alt:10px, sağ-üst/sol-alt:20px */
border-radius: 50%;          /* Tam daire */

📐 4. MARGIN — Dış Boşluk

Margin, bu kutunun diğer kutularla arasındaki dış boşluktur. Margin, elementin dışına eklenir ve diğer elementleri iter.

/* Margin kullanımı */
margin: 10px;               /* 4 yönde de 10px */
margin: 10px 20px;          /* üst/alt:10px, sağ/sol:20px */
margin: 10px 20px 30px;     /* üst:10px, sağ/sol:20px, alt:30px */
margin: 10px 20px 30px 40px; /* üst, sağ, alt, sol */

/* Tek yönlü margin */
margin-top: 10px;
margin-right: 20px;
margin-bottom: 30px;
margin-left: 40px;

/* Otomatik margin (ortalamak için) */
margin: 0 auto;             /* Yatayda ortalama */

⚠️ Margin Çökmesi (Margin Collapse): İki elementin dikey margin'leri birleştiğinde, en büyük olan geçerli olur, toplanmaz! 
Kutu A: margin-bottom: 30px;
Kutu B: margin-top: 20px;
→ Aralarındaki boşluk 50px DEĞİL, 30px olur!

🧮 Toplam Kutu Boyutu Nasıl Hesaplanır?

Varsayılan olarak (box-sizing: content-box):

Toplam Genişlik = width + padding-left + padding-right + border-left + border-right + margin-left + margin-right
Toplam Yükseklik = height + padding-top + padding-bottom + border-top + border-bottom + margin-top + margin-bottom
📊 Örnek Hesaplama: 
.kutu {
  width: 200px;
  padding: 20px;
  border: 5px solid black;
  margin: 10px;
}

Toplam Genişlik = 200 + 20 + 20 + 5 + 5 + 10 + 10 = 270px!

⚙️ box-sizing — Hayat Kurtaran Özellik

box-sizing özelliği, width ve height'in nasıl hesaplanacağını belirler. Bu, CSS'deki en önemli özelliklerden biridir!

/* content-box (Varsayılan) */
box-sizing: content-box;
/* width ve height SADECE content alanını belirler */
/* Padding ve border EKLENİR */

/* border-box (ÖNERİLEN) */
box-sizing: border-box;
/* width ve height content + padding + border'ı içerir */
/* Padding ve border EKSTRA ALAN EKLEMEZ! */
🎯 border-box neden daha iyi?

Border-box ile kutunun tam boyutunu kontrol edebilirsiniz. Padding veya border eklediğinizde kutu büyümez, içerik alanı küçülür.

📺 content-box vs border-box

content-box (Varsayılan)
200px width

Toplam genişlik: 250px (200 + 40 + 10)

border-box (Önerilen)
200px width

Toplam genişlik: 200px (padding ve border içeriye alınır)

💡 En iyi pratik — Tüm elementlere border-box uygulayın: 
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

Bu, tüm elementlerin daha öngörülebilir boyutlarda olmasını sağlar!

⚠️ Sık Yapılan Hatalar ve Çözümleri

❌ Hata 1: Padding ve border'ın toplam genişliğe eklendiğini unutmak 
.kutu {
  width: 200px;
  padding: 20px;   /* Toplam 240px olur! */
}
✅ Çözüm: box-sizing: border-box; kullanın!
❌ Hata 2: Margin çökmesini (collapse) bilmemek 
.kutu1 { margin-bottom: 30px; }
.kutu2 { margin-top: 20px; }
/* Aralarındaki boşluk 50px DEĞİL, 30px! */
✅ Çözüm: Sadece bir yönde margin kullanın (örn: sadece margin-bottom).
❌ Hata 3: İnline elementlere margin/padding vermek 
span {
  margin: 20px;  /* Sadece sağ/sol çalışır, üst/alt çalışmaz! */
  padding: 20px; /* Çalışır ama diğer elementleri etkilemez */
}
✅ Çözüm: display: inline-block; veya display: block; kullanın.
❌ Hata 4: Yüzdesel margin/padding'in ebeveyne göre hesaplandığını unutmak 
.parent { width: 500px; }
.child { padding: 10%; }  /* 500px'in %10'u = 50px */
✅ Çözüm: Yüzdesel değerlerin her zaman ebeveynin genişliğine göre hesaplandığını unutmayın!

📋 Box Model Özeti

  • 📦 Her HTML elementi bir kutudur — Content, Padding, Border, Margin
  • 📄 Content — İçerik alanı (width/height burayı belirler)
  • 📏 Padding — İç boşluk (içerik ile border arası)
  • 🖌️ Border — Kenarlık (stil, kalınlık, renk)
  • 📐 Margin — Dış boşluk (diğer kutularla arası)
  • 🧮 Toplam genişlik = width + padding + border + margin (content-box'ta)
  • ⚙️ box-sizing: border-box — Tüm elementlere uygulayın! (Hayat kurtarır)
  • ⚠️ Margin collapse — Dikey margin'ler birleşir, toplanmaz
