📦 <div> ve <span>

HTML'in En Temel Kapsayıcı Elementleri — Gruplama ve Stil Verme

🎯 Giriş: Neden Div ve Span?

<div> ve <span>, HTML'in en çok kullanılan elementleridir. Tek başlarına hiçbir anlamları yoktur, ancak diğer elementleri gruplamak ve onlara CSS veya JavaScript ile müdahale etmek için kullanılırlar.

💡 Altın Kural: Div ve span'ın tek amacı kapsayıcı (container) olmaktır. Onlar olmadan sayfalarımızı düzenlemek çok zor olurdu!

🔷 <div> — Block Seviye Kapsayıcı

📦 Division / Bölüm
<div> display: block (varsayılan)

<div> (division), block-level bir elementtir. Sayfada yeni bir satır başlatır ve bulunduğu kabın tüm genişliğini kaplar.

<div class="kutu">
  <h1>Başlık</h1>
  <p>Bu bir paragraf. Div içinde gruplanmış içerik.</p>
  <button>Tıkla</button>
</div>

<div class="kutu">
  <p>İkinci div yeni satırda başlar</p>
</div>

📺 Div ile Gruplama Örneği

🔹 1. Div: İçinde her türlü element olabilir
🔹 2. Div: Her div yeni satırda başlar
🔹 3. Div: Tam genişliği kaplar (kırmızı çizgi border)

🎨 Div ile Sayfa Düzeni Oluşturma

<div class="page-layout">
  <div class="header">Üst Menü</div>
  <div class="nav">Navigasyon</div>
  <div class="main">
    <div class="content">Ana İçerik</div>
    <div class="sidebar">Yan Menü</div>
  </div>
  <div class="footer">Alt Bilgi</div>
</div>

📺 Div ile Yapılmış Sayfa Düzeni

📰 HEADER (Üst Bölüm)
🔗 NAV (Menü Bağlantıları)
MAIN CONTENT (Ana İçerik)
Burası div ile gruplanmış ana içerik alanı. Blog yazıları, makaleler burada olabilir.
SIDEBAR (Yan Menü)
Kategoriler, popüler yazılar, reklamlar...
© FOOTER (Alt Bilgi) — Tüm hakları saklıdır.
✅ Div Kullanım Alanları:
  • Sayfa bölümlerini ayırmak (header, main, footer, sidebar)
  • Form grupları oluşturmak
  • CSS Grid veya Flexbox ile düzen oluşturmak
  • JavaScript ile müdahale edilecek alanları belirlemek
  • Kartlar, galeriler, modal pencereler gibi UI bileşenleri yapmak

🔹 <span> — Inline Seviye Kapsayıcı

📝 Span / Aralık
<span> display: inline (varsayılan)

<span>, inline-level bir elementtir. Yeni satır başlatmaz, sadece içeriği kadar yer kaplar. Genellikle metin içindeki küçük parçaları stillendirmek için kullanılır.

<p>
  Bugün <span class="highlight">HTML</span> öğreniyoruz.
  <span class="important">Div ve span</span> çok önemli!
</p>

📺 Span ile Metin İçi Stilleme

Bugün HTML öğreniyoruz. Div ve span çok önemli elementlerdir. Ayrıca inline elementlerdir.

🎨 Span ile Detaylı Örnek

<p>
  Bugün <span style="color: red;">hava</span> çok güzel.
  Fiyat: <span style="font-weight: bold; color: green;">$49.99</span>
  <span style="text-decoration: line-through;">$99.99</span>
</p>

📺 Span ile Detaylı Stilleme

Bugün hava çok güzel.
Fiyat: $49.99 $99.99
Durum: Stokta var

📌 Span Kullanım Alanları:
  • Metin içinde belirli kelimeleri renklendirmek
  • Fiyat etiketleri, indirim bilgileri göstermek
  • Icon veya küçük elementleri metinle aynı satırda göstermek
  • JavaScript ile metin içindeki belirli alanları yakalamak
  • Tooltip, badge gibi küçük UI elementleri oluşturmak
⚠️ ÖNEMLİ: Span inline element olduğu için width ve height özellikleri ÇALIŞMAZ. Genişlik/yükseklik vermek istersen display: inline-block; kullanmalısın!

🔄 Display Özelliği ile Tip Değiştirme

CSS'in display özelliği ile bir elementin görüntüleme tipini değiştirebilirsiniz!

/* Span'i block gibi davranmaya zorla */
span {
  display: block;
}

/* Div'i inline gibi davranmaya zorla */
div {
  display: inline;
}

/* Her ikisini de inline-block yap */
span, div {
  display: inline-block;
}

📺 Span display: block yapıldı

Bu span normalde inline ama display: block yapıldı İkinci span da block, yeni satırda başlıyor!

📺 Div display: inline yapıldı

⬆️ Normalde her div yeni satırda başlardı, ama display: inline ile yan yana dizildiler!

⚠️ En Sık Yapılan Hatalar

❌ Hata 1: Span'e width/height vermeye çalışmak 
span {
  width: 200px;   /* ❌ ÇALIŞMAZ! Span inline elementtir */
  height: 100px;  /* ❌ ÇALIŞMAZ! */
}
✅ Çözüm: display: inline-block; ekleyin!
❌ Hata 2: Div içine sadece inline element koyup stillendirmeye çalışmak 
<div>
  <span>Metin</span>
  <span>Metin</span>
</div>
/* Div'e text-align: center yaparsan içindeki span'ler ortalanır */
✅ Çözüm: Div block olduğu için içindeki inline elementleri hizalamak için div'e text-align verebilirsiniz.
❌ Hata 3: Anlamsal elementler yerine sürekli div kullanmak 
<div class="header">...</div>  ⚠️ Daha iyisi: <header>...</header>
<div class="nav">...</div>     ⚠️ Daha iyisi: <nav>...</nav>
<div class="footer">...</div>  ⚠️ Daha iyisi: <footer>...</footer>
✅ Çözüm: HTML5'te anlamsal elementler (header, nav, main, article, section, footer) varken, anlamlı bir element yoksa div kullanın!

📋 Özet

  • 📦 <div> = Block element → Büyük blokları gruplamak için (sayfa düzeni, kartlar, formlar)
  • 📝 <span> = Inline element → Metin içindeki küçük parçaları gruplamak için (renk, vurgu, fiyat)
  • 🎯 Div ve span'ın tek amacı kapsayıcı (container) olmaktır
  • 🔄 display özelliği ile tiplerini değiştirebilirsiniz
  • ⚠️ Span'e width/height vermek için display: inline-block kullanın
  • 🏷️ Anlamlı bir element yoksa div/span kullanın (header, nav, footer gibi elementler varken onları tercih edin)
