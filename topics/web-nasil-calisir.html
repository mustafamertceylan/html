🌐 Web Nasıl Çalışır?

Browsers · Internet · Web Servers · HTML

🖥️ Browsers! (Tarayıcılar)

Tarayıcılar, web sayfalarını gösterebilen uygulamalardır. İnternete girdiğinizde kullandığınız her araç bir tarayıcıdır.

Chrome Firefox Safari Edge Internet Explorer Opera

Tarayıcıların görevi: HTML okuyup yorumlamak ve ekrana görsel olarak çizmek.

📝 Web Sayfaları HTML ile Yazılır

Web sayfaları HTML adında bir işaretleme diliyle yazılır. Tarayıcılar, bir web sayfasını göstermek için HTML kodunu okur ve yorumlar.

┌─────────────────────────────────────────┐
│  HTML Kodu (örnek)                       │
│  <h1>Merhaba Dünya!</h1>                │
│  <p>Bu bir paragraf.</p>                │
└─────────────────────────────────────────┘
                    ↓
              Tarayıcı okur & yorumlar
                    ↓
┌─────────────────────────────────────────┐
│  Ekranda Görünen                         │
│                                          │
│     Merhaba Dünya!                       │
│     Bu bir paragraf.                     │
└─────────────────────────────────────────┘

🔗 HTML Başka Kaynaklara Link Verir

Bir HTML dosyası, diğer kaynaklara (resimler, videolar, CSS stilleri, JavaScript dosyaları) link (bağlantı) verebilir. Tarayıcı bu kaynakları da ayrıca yükler.

HTML dosyası içinde:
─────────────────────────────────────────────────
  <img src="foto.jpg">        → resim yüklenir
  <link rel="stylesheet" href="style.css"> → CSS yüklenir
  <script src="kod.js"></script> → JavaScript yüklenir
  <video src="film.mp4"></video> → video yüklenir
─────────────────────────────────────────────────

🗄️ Web Server (Web Sunucusu)

Bir web sunucusu, bir bilgisayarda çalışan ve isteklere yanıt olarak web sayfalarını teslim eden bir programdır. Sayfaları ya saklar (statik) ya da anında oluşturur (dinamik).

┌──────────────┐      İstek (Request)      ┌──────────────┐
│   Tarayıcı   │ ───────────────────────→  │  Web Sunucusu │
│   (Browser)  │                           │  (Server)     │
│              │ ←───────────────────────  │              │
└──────────────┘      Cevap (Response)     └──────────────┘
                      (HTML dosyası)

⚙️ Adım Adım Web Sayfası Yüklenmesi

1
URL girilir

Tarayıcıya bir URL (adres) yazarsınız. Bu, internetteki HTML dosyasının adresidir.

cs193x.stanford.edu
2
Tarayıcı, web sunucusuna sorar

Tarayıcı, o belgeyi barındıran web sunucusuna "Bu belgeyi gönder" diye bir istek yapar.

📤 GET /index.html → Tarayıcı → Sunucu: "HTML dosyasını ister"
3
Web sunucusu cevap verir

Sunucu, tarayıcıya istenen HTML dosyasını gönderir.

📥 OK + HTML içeriği ← Sunucu → Tarayıcı: "İşte HTML dosyan"
4
Tarayıcı gömülü kaynakları da ister

Tarayıcı HTML'i okur, gömülü kaynakları (resim, CSS, JS) görür ve sunucudan onları da ister.

📤 GET /style.css 📥 OK + CSS
📤 GET /script.js 📥 OK + JavaScript
📤 GET /image.png 📥 OK + Resim
5
Sayfa yüklenir ve görüntülenir

Tüm kaynaklar (HTML, CSS, JS, resimler, videolar) getirildiğinde ve tarayıcı tarafından yorumlandığında, web sayfası tamamen görüntülenir.

✨ Sayfa hazır! ✨
