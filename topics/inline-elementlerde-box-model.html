📦 Inline Elementlerde Box Model

Inline elementlerde margin, padding, line-height ve farklılıklar

🔍 Inline Elementlerde Box Model Nasıl Çalışır?

Soru: Box model inline elementler için de geçerli mi?

Cevap: Evet, ancak kutu farklı şekillenir!

📌 Inline Elementlerde Box Model Farkları:
  • ✅ margin-left ve margin-right → ÇALIŞIR (diğer elementleri iter)
  • ❌ margin-top ve margin-bottom → ÇALIŞMAZ (diğer elementleri itmez)
  • ✅ padding-left ve padding-right → ÇALIŞIR
  • ⚠️ padding-top ve padding-bottom → ÇALIŞIR AMA diğer elementleri etkilemez (taşabilir)
  • ✅ line-height → Satırlar arası mesafeyi ayarlamak için KULLANILIR
<span>Bir inline element</span>
<span>Başka bir inline element</span>

span {
  margin: 20px;        /* margin-top/bottom ÇALIŞMAZ! */
  margin-left: 20px;   /* ✅ ÇALIŞIR */
  margin-right: 20px;  /* ✅ ÇALIŞIR */
  padding: 10px;       /* padding-top/bottom çalışır ama diğerlerini itmez */
  line-height: 2;      /* ✅ Satır yüksekliğini ayarlar */
}

📏 line-height — Satır Yüksekliği

line-height, inline elementlerde satırlar arasındaki dikey boşluğu kontrol etmek için kullanılır. Block elementlerde de çalışır, ancak inline elementler için çok daha kritiktir.

/* line-height kullanımı */
line-height: 1.5;     /* Yazı boyutunun 1.5 katı */
line-height: 24px;    /* Sabit piksel değeri */
line-height: 150%;    /* Yüzdesel değer */
line-height: normal;  /* Varsayılan (yaklaşık 1.2) */

💡 line-height nerede kullanılır?
  • Paragraflar arasındaki okunabilirliği artırmak için
  • Metni dikeyde ortalamak için (inline-block ile birlikte)
  • Butonlarda metni dikeyde ortalamak için

⚠️ body'nin Varsayılan Margin'i

Tarayıcılar, <body> elementine varsayılan olarak margin uygular. Bu nedenle elementler sayfanın tam kenarına yapışmaz!

📺 Varsayılan body margin'i

Bu içerik body içinde. Beyaz boşluk = body'nin varsayılan margin'i! 
/* Varsayılan margin'i kaldır */
body {
  margin: 0;
  padding: 0;
}
⚠️ Önemli: Çoğu tarayıcıda body'nin varsayılan margin değeri 8px'dir. Sayfanı sıfırdan tasarlarken mutlaka margin: 0; ekleyin!

🎯 Layout Çalışması — "Burada Takıldık"

Şimdiye kadar öğrendiklerimizle bir Squarespace benzeri template yapmaya çalışıyoruz:

Navbar:     height: 75px;     background: royalblue;    <nav>
Header:     height: 400px;    background: lightskyblue; <header>
Section:    height: 500px;    background: gray;         <section>
Footer:     height: 100px;    background: black;        <footer>
🤔 Peki Nerede Takılıyoruz?
  • İçerikte iki kutuyu yan yana nasıl koyacağız? (%48 genişlik, aralarında boşluk)
  • Kutuları yatayda ortalama nasıl yapacağız?
  • Elementler arasındaki boşlukları nasıl kontrol edeceğiz?
💡 Çözüm: Bu sorunları çözmek için CSS Flexbox veya CSS Grid kullanacağız! Bunlar bir sonraki konularımız olacak.

📋 Özet

  • 📦 Inline elementlerde margin — sadece sağ/sol çalışır, üst/alt ÇALIŞMAZ
  • 📏 line-height — satırlar arası mesafeyi ayarlar, inline elementler için kritiktir
  • ⚠️ body'nin varsayılan margin'i — sayfa kenarında boşluk bırakır, margin: 0; ile kaldırılır
  • 🏷️ Semantic HTML — <div> yerine anlamlı elementler kullanın (header, nav, main, section, article, footer)
  • 🎯 Layout sorunları — Yan yana kutu yerleştirmek için Flexbox veya Grid kullanacağız
