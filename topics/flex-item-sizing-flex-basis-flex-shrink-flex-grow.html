📏 Flex Item Sizing

flex-basis · flex-shrink · flex-grow · align-items: stretch — Flex item'ların boyutlandırılması

📐 Flex Basis — Flex Item'ın Başlangıç Boyutu

Flex basis, bir flex item'ın başlangıç genişliğidir (flex-direction: row iken genişlik, column iken yükseklik). Flex item'ın büyüme/küçülme uygulanmadan önceki varsayılan boyutudur.

📌 Flex Basis Nasıl Belirlenir? (Öncelik sırası)
  1. flex-basis özelliği (en yüksek öncelik)
  2. width özelliği (flex-basis yoksa)
  3. İçerik genişliği (content width) — hiçbiri yoksa
/* Flex Basis Örnekleri */
.item1 {
  flex-basis: 200px;  /* Başlangıç genişliği 200px */
}

.item2 {
  width: 150px;       /* flex-basis yoksa width kullanılır */
}

.item3 {
  /* Hiçbir şey belirtilmemiş → içerik genişliği */
}

📺 Flex Basis Örneği

flex-basis: 200px
width: 150px
İçerik genişliği

⬆️ Her flex item farklı başlangıç genişliğine sahip!

💡 Önemli: Flex item'ların width özelliği, normal block/inline elementlerden farklı olarak her zaman saygı görür — inline olsalar bile!

📉 flex-shrink — Flex Item Küçülmesi

flex-shrink, flex item'ın container'dan küçüldüğünde ne kadar küçüleceğini belirler. Varsayılan değer 1'dir.

flex-shrink: 0;  /* Hiçbir zaman küçülmez! */
flex-shrink: 2;  /* Diğerlerinden 2 kat fazla küçülür */

📏 align-items: stretch — Varsayılan Dikey Gerilme

align-items: stretch, flex container'ın varsayılan değeridir. Bu sayede flex item'lar dikeyde (row iken) container'ın yüksekliğine gerilir.

  height: 150px;

⚡ flex Kısayolu — flex: grow shrink basis

flex özelliği, flex-grow, flex-shrink ve flex-basis'i tek satırda yazmak için kullanılır.

/* Kısayol kullanımı */
flex: 1;           /* flex-grow: 1, flex-shrink: 1, flex-basis: 0 */
flex: auto;        /* flex-grow: 1, flex-shrink: 1, flex-basis: auto */
flex: none;        /* flex-grow: 0, flex-shrink: 0, flex-basis: auto */
flex: 2 1 200px;   /* grow:2, shrink:1, basis:200px */

/* En yaygın kullanım — Eşit genişlikli sütunlar */
.item {
  flex: 1;  /* Tüm item'lar eşit genişlikte büyür */
}

🔄 flex-direction: column için Boyutlandırma

flex-direction column iken, genişlik ve yükseklik rolleri değişir:

  • flex-basis → YÜKSEKLİK (başlangıç yüksekliği)
  • flex-grow → Dikeyde büyüme (yükseklik)
  • flex-shrink → Dikeyde küçülme (yükseklik)
  • align-items: stretch → YATAYDA gerilme (genişlik)
.container {
  display: flex;
  flex-direction: column;
  height: 300px;
}

.item {
  flex: 1;  /* Dikeyde eşit yükseklik dağılımı! */
}

⚠️ Sık Yapılan Hatalar

❌ Hata 1: flex-grow ve width birlikte kullanıldığında karışıklık 
.item {
  width: 200px;
  flex-grow: 1;  /* width + flex-grow birlikte çalışır */
}
✅ Çözüm: width başlangıç değeridir, flex-grow buna eklenir.
❌ Hata 2: flex-shrink: 0 kullanmayı unutmak 
.sidebar {
  width: 250px;
  /* flex-shrink: 0; yok → daralmaya izin verir! */
}
✅ Çözüm: Sabit genişlikli elementlere flex-shrink: 0 ekleyin!
❌ Hata 3: align-items: stretch'in height ile ezildiğini unutmak 
.item {
  height: 100px;  /* stretch çalışmaz! */
}
✅ Çözüm: stretch istiyorsanız height belirtmeyin.

📋 Flex Item Sizing — Özet

  • 📐 flex-basis → Başlangıç boyutu (row: genişlik, column: yükseklik)
  • 📉 flex-shrink: 1 (varsayılan) → Alan daraldığında küçülür
  • 📈 flex-grow: 0 (varsayılan) → Alan genişlediğinde büyümez
  • ⚡ flex: 1 → grow:1, shrink:1, basis:0 (eşit dağılım için)
  • 📏 align-items: stretch (varsayılan) → Dikeyde gerilme
  • 🔄 flex-direction: column → Eksenler değişir (grow dikeyde çalışır)
  • 🔒 Sabit genişlikli elementler için flex-shrink: 0 kullanın
