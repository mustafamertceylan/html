CSS Flexbox — Derinlemesine (Detaylı)

Flexbox modern CSS layout modülü. Flex Container ve Flex Item yapısı, justify-content (flex-start, center, space-between, space-around), align-items (stretch, flex-start, center, flex-end), flex-direction (row, column, reverse), flex-grow/shrink/basis, inline elementlerin flex item olunca davranışı ve pratik örnekler.

💪 CSS Flexbox

Modern web düzenlerinin vazgeçilmezi — Flex Container · Flex Items · justify-content · align-items · flex-direction

📐 CSS Layout — Bugüne Kadar Ne Öğrendik?

Şimdiye kadar iki temel layout mekanizması öğrendik:

1. Block Layout (Block Düzeni)

Sayfanın büyük bölümlerini yerleştirmek için kullanılır. Elementler yukarıdan aşağıya (top-to-bottom) akar.

<div>Block element 1</div>
<div>Block element 2</div>
<div>Block element 3</div>

/* Her div yeni satırda başlar, tam genişlik kaplar */
2. Inline Layout (Inline Düzeni)

Metin ve diğer inline içerikleri bir bölüm içinde yerleştirmek için kullanılır. Elementler soldan sağa (left-to-right) akar.

<span>Inline element 1</span>
<span>Inline element 2</span>
<span>Inline element 3</span>

/* Tüm span'ler yan yana dizilir, sadece içerik kadar yer kaplar */
💡 Peki ya daha karmaşık düzenler?

Block ve inline layout ile yapması çok zor olan bazı düzenler vardır:

  • Elementleri yatayda ortalamak (margin: 0 auto çalışır ama sınırlıdır)
  • Elementleri dikeyde ortalamak (block layout'ta çok zordur!)
  • Elementleri eşit aralıklarla yerleştirmek
  • Elementleri ters sırada göstermek
  • Farklı boyutlardaki elementleri esnek şekilde hizalamak
🎯 İşte burada FLEX LAYOUT devreye giriyor!

Flex layout, bu sorunları zarif ve kolay bir şekilde çözer.

🔧 Flex Layout Nedir?

Flex layout, elementleri satır (row) veya sütun (column) halinde düzenlemek için özel kurallar tanımlayan bir CSS layout modudur.

🎯 Flex Layout ile KOLAYCA yapılabilenler:
  • ✅ Navigasyon menüleri (yan yana, eşit aralıklı)
  • ✅ Kart galerileri (responsive, esnek)
  • ✅ Sayfa içeriğini yatayda ve dikeyde ortalamak
  • ✅ Footer'ı sayfanın altına yapıştırmak
  • ✅ Elementleri ters sırada göstermek
  • ✅ Farklı boyutlardaki elementleri hizalamak
📌 Flex Layout'un Temel Yapısı: 
┌─────────────────────────────────────────────────────────┐
│                   FLEX CONTAINER                         │
│  ┌─────────────┐ ┌─────────────┐ ┌─────────────┐        │
│  │ FLEX ITEM 1 │ │ FLEX ITEM 2 │ │ FLEX ITEM 3 │        │
│  └─────────────┘ └─────────────┘ └─────────────┘        │
│                                                         │
└─────────────────────────────────────────────────────────┘

- Flex Container: display: flex veya inline-flex ile oluşturulur
- Flex Item: Container'ın direkt çocuk elementleri

📦 Flex Container Oluşturma

Bir elementi flex container yapmak için display: flex; veya display: inline-flex; kullanılır.

/* Block-level flex container (tam genişlik) */
.container {
  display: flex;
}

/* Inline-level flex container (içerik kadar genişlik) */
.container {
  display: inline-flex;
}

🤔 Peki Ya Inline Elementler?

Hatırlayalım: Normalde <span> gibi inline elementler width ve height özelliklerine sahip değildir.

<span>Bu bir inline element</span>
<span>width ve height çalışmaz!</span>
❌ Normal block layout'ta:
Span 1 Span 2 Span 3

⬆️ Bu span'ler inline olduğu için width/height ayarlanamaz, yan yana dizilirler.

✅ Flex container içinde:

Bir elementi display: flex yaptığınızda, direkt çocuk elementler artık inline veya block değil, FLEX ITEM olurlar. Flex item'ler farklı kurallara sahiptir!

Span 1 (flex item) Span 2 (flex item) Span 3 (flex item)

⬆️ Aynı span elementleri, ama flex container içinde oldukları için genişlikleri ayarlanabilir ve esnek davranırlar!

💡 Neden böyle?
  • Flex container, direkt çocuklarını flex item haline getirir
  • Flex item'ler, block veya inline değildir; farklı kuralları vardır
  • Flex item'lerin height ve width özellikleri çalışır
  • Flex item'lerin içindeki içerikler, flex item'ın sınırlarına göre normal block/inline kurallarını takip eder

📦 Flex Item Özellikleri

📌 flex-grow — Büyüme Oranı

Bir flex item'in, kalan boş alanı ne kadar alacağını belirler.

.item1 { flex-grow: 1; }
.item2 { flex-grow: 2; }  /* item1'den 2 kat fazla büyür */
.item3 { flex-grow: 1; }
flex-grow: 1
flex-grow: 2
(2 kat büyük)
flex-grow: 1

📌 flex-shrink — Küçülme Oranı

Bir flex item'in, alan daraldığında ne kadar küçüleceğini belirler.

📌 flex-basis — Başlangıç Boyutu

Bir flex item'in, büyüme/küçülme uygulanmadan önceki varsayılan boyutu.

📌 flex (Kısayol)

flex: 1;           /* flex-grow: 1, flex-shrink: 1, flex-basis: 0 */
flex: 2 1 100px;   /* grow, shrink, basis */

📌 align-self — Bireysel Hizalama

Bir flex item'in, container'daki align-items ayarını ezmesini sağlar.

align-items: center
align-self: flex-start
align-self: flex-end

📋 Flexbox Özeti

  • 🎯 Flexbox, karmaşık düzenleri kolayca oluşturmak için bir CSS layout modudur
  • 📦 Flex Container → display: flex veya inline-flex
  • 📄 Flex Item → Container'ın direkt çocukları
  • ↔️ justify-content → Ana eksende hizalama (row: yatay, column: dikey)
  • ↕️ align-items → Çapraz eksende hizalama (row: dikey, column: yatay)
  • 🔄 flex-direction → Yön değiştirme (row, column, row-reverse, column-reverse)
  • 📏 flex-grow / shrink / basis → Flex item'lerin boyut davranışı
  • ✨ Flex item'ler inline olsalar bile width/height alabilir!
  • 🎯 justify-content + align-items: center → Mükemmel ortalama!
