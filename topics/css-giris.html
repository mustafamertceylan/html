🎨 CSS — Cascading Style Sheets

Stil kuralları · Seçiciler · Renkler · HTML'e bağlama · Alıştırma

📖 CSS Nedir?

CSS (Cascading Style Sheets) bir web sayfasının görünümünü ve düzenini tanımlar.

CSS, stil kurallarından (style rules) oluşur. Her kural, hangi HTML elementlerinin nasıl görüneceğini belirtir.

selector {
  property: value;
}

/* Örnek: Tüm paragraflar mavi ve kalın olsun */
p {
  color: blue;
  font-weight: bold;
}

✨ Canlı Önizleme (CSS uygulanmış)

Bu paragraf mavi ve kalın!

🔧 CSS Kuralının Yapısı

selector {
  property: value;
}
  • Seçici (selector): Hangi HTML element(ler)inin stil alacağını belirtir.
  • Özellik (property): CSS stilinin adı (ör: color, font-size).
  • Değer (value): CSS stili için değer (ör: blue, 16px).
💡 Benzetme: fork { color: gold; } → "Masamdaki tüm çatallar altın rengi olsun" gibi düşünebilirsin.

🔗 CSS Dosyası ve HTML'e Bağlama

CSS kuralları style.css gibi bir dosyada saklanır. HTML dosyasında <link> etiketi ile bağlanır:

<!DOCTYPE html>
<html>
<head>
  <title>CS 193X</title>
  <link rel="stylesheet" href="filename.css" />
</head>
<body>
  ... sayfa içeriği ...
</body>
</html>
📌 Önemli: link etiketi <head> içinde olmalıdır. href özniteliği CSS dosyasının yolunu belirtir.

🧠 Önemli Teknikler & Karşılaşılan Tuhaf Durumlar

✅ İyi Bilinen Teknikler

  • Görünmez kaplar (containers) ekleyin: Element gruplarını seçmek için <div> veya <span> kullanın.
  • Üst/ata elemente stil uygulayın: Üst elemente stil vererek tüm çocuk elementleri de stillendirebilirsiniz.

⚠️ Karşılaşılan Tuhaf Durumlar

  • text-align: center; → <a> veya <strong> gibi inline elementlerde çalışmaz. Onları display: block; yapmalısınız.
  • width belirlemek: Kutu içeriğine tam otursun istiyorsan width: fit-content; veya display: inline-block; kullanabilirsin.
  • Kutu ortalama: margin: 0 auto; sadece width belirtilmiş block elementlerde çalışır.
