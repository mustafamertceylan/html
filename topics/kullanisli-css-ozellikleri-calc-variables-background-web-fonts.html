🧮 calc() — Dinamik Hesaplamalar

calc() fonksiyonu, CSS'de matematiksel ifadeler kullanarak değer hesaplamanızı sağlar. Farklı birimleri (px, %, em, vh, vw) birbiriyle toplayabilir, çıkarabilir, çarpabilir veya bölebilirsiniz.

/* calc() kullanım örnekleri */
width: calc(50% - 10px);      /* Yüzde 50 eksi 10 piksel */
width: calc(100% / 3);        /* %100'ün 3'te 1'i */
width: calc(100vw - 80px);    /* Tam viewport genişliği eksi 80px */
font-size: calc(1rem + 2vw);  /* Responsive font */

🎨 CSS Variables (Özel Değişkenler)

CSS Variables (CSS Custom Properties), tüm CSS dosyanızda yeniden kullanılabilir değerler tanımlamanızı sağlar. Değişkenler -- ile başlar ve var() fonksiyonu ile kullanılır.

/* Değişken tanımlama — genellikle :root içinde */
:root {
  --primary-color: #e8ff47;
  --spacing: 1rem;
  --border-radius: 12px;
}

/* Değişken kullanma */
.button {
  background: var(--primary-color);
  padding: var(--spacing);
  border-radius: var(--border-radius);
}

Değişkenlerle Stillenmiş Kart

Bu kartın renkleri, boşlukları ve border-radius'u CSS değişkenleri ile kontrol ediliyor.

🎯 CSS Variables Avantajları:
  • ✅ Tekrar Kullanılabilirlik — Aynı değeri bir yerde tanımla, her yerde kullan
  • ✅ Kolay Bakım — Tema rengini değiştirmek için tek bir yeri değiştir
  • ✅ Dinamik — JavaScript ile değiştirilebilir
  • ✅ Scope (Kapsam) — Belirli bir elementin içinde de tanımlanabilir
📌 JavaScript ile Değişken Değiştirme: 
// Tüm sayfanın ana rengini değiştir
document.documentElement.style.setProperty('--primary-color', '#ff0000');

// Belirli bir elementin değişkenini değiştir
element.style.setProperty('--spacing', '2rem');

🖼️ Background Properties — Arka Plan Özellikleri

Bir elemente background-image ekleyerek, resimleri kolayca yerleştirebilir ve boyutlandırabilirsiniz.

.hero {
  background-image: url('background.jpg');
  background-size: cover;      /* Resmi tam kapla, kırp */
  background-position: center; /* Ortala */
  background-repeat: no-repeat; /* Tekrar etme */
  height: 400px;
}

📌 background-size: cover (Tam kapla, kırparak)

background-size: cover

📌 background-size: contain (Tam sığdır, boşluk bırakabilir)

📌 Önemli Background Özellikleri:
background-size: cover Resmi tam kaplar, taşan kısımlar kırpılır
background-size: contain Resmi tam sığdırır, boşluk kalabilir
background-position: center Resmi ortalar (top, bottom, left, right da kullanılabilir)
background-repeat: no-repeat Resmin tekrar etmesini engeller

🔤 Web Fonts — Google Fonts ile Daha İyi Fontlar

Google Fonts, web projelerinizde kullanabileceğiniz ücretsiz ve kaliteli fontlar sunar.

<!-- HTML head bölümüne ekle -->
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;700&display=swap" rel="stylesheet">

<!-- CSS'te kullan -->
body {
  font-family: 'Inter', sans-serif;
}

📌 Fallback Fontlar (Yedek Fontlar)

font-family'de virgülle ayrılmış birden fazla font belirtebilirsiniz. İlk font yüklenemezse, sonraki font kullanılır.

/* Önce Inter dene, yoksa Arial, o da yoksa sans-serif kullan */
font-family: 'Inter', Arial, sans-serif;

/* Generic font ailesi (sans-serif, serif, monospace, cursive) her zaman çalışır! */

Bu font önce NonExistentFont'u dener, bulamazsa Arial, o da yoksa sans-serif kullanır.

📁 @font-face — Kendi Fontunu Yükle

@font-face kuralı ile, kendi sunucunuzda barındırdığınız özel font dosyalarını yükleyebilirsiniz.

/* CSS dosyasında font tanımlama */
@font-face {
  font-family: 'MyCustomFont';
  src: url('fonts/myfont.woff2') format('woff2'),
       url('fonts/myfont.woff') format('woff');
  font-weight: normal;
  font-style: normal;
}

/* Fontu kullanma */
.custom-text {
  font-family: 'MyCustomFont', sans-serif;
}
🎯 @font-face için dosya formatları:
  • .woff2 — En iyi sıkıştırma, modern tarayıcılar (önerilen)
  • .woff — Geniş tarayıcı desteği
  • .ttf / .otf — TrueType/OpenType (daha büyük dosya boyutu)
  • .eot — Eski Internet Explorer için
💡 Google Fonts vs @font-face:
  • Google Fonts → Kolay, hızlı, ücretsiz, geniş seçenek
  • @font-face → Tam kontrol, özel/kurumsal fontlar, gizlilik
