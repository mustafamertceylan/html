CSS Position — Static, Fixed, Absolute, Relative (Detaylı)

CSS position özelliği ile elementleri tam istediğin yere konumlandır. Static (varsayılan), Fixed (viewport'a sabit), Absolute (ebeveyne göre mutlak), Relative (normal konumundan kaydırma), z-index ile katman sıralaması. Overlay, sabit navbar, kart üzeri etiket, modal gibi pratik örnekler ve sık yapılan hatalar.

📍 CSS Position

static · fixed · absolute · relative · z-index — Elementleri tam istediğin yere koy!

🎯 Position Nedir?

Positioned layout, bir elementin sayfada tam olarak nerede olacağını belirlemenizi sağlar.

📌 Position Kullanmak İçin 3 Adım:
  1. Position method belirle: static, fixed, absolute, relative
  2. Offset (uzaklık) belirle: top, left, bottom, right
  3. z-index (isteğe bağlı) → Katman sıralaması
/* Position örnek yapısı */
.element {
  position: fixed;     /* 1. Position method */
  top: 50px;           /* 2. Offset (uzaklık) */
  left: 100px;
  z-index: 10;         /* 3. Katman sırası (isteğe bağlı) */
}

📌 position: static — Varsayılan Değer (Hiçbir Şey Olmaz!)

position: static, tüm elementlerin varsayılan değeridir. Bu modda elementler normal belge akışında (normal flow) kalır ve top/left/bottom/right özellikleri ÇALIŞMAZ!

.static-box {
  position: static;    /* Varsayılan - zaten gerekmez */
  top: 50px;           /* ❌ ÇALIŞMAZ! */
  left: 100px;         /* ❌ ÇALIŞMAZ! */
}

⚠️ Önemli: top/left/bottom/right kullanabilmek için position static OLMAMALI!

📌 position: fixed — Viewport'a Sabit

position: fixed ile bir element, viewport'a göre sabitlenir ve sayfa kaydırıldığında hareket etmez. Element normal akıştan çıkarılır ve kendi katmanında konumlandırılır.

.fixed-menu {
  position: fixed;
  top: 20px;
  right: 20px;
  background: #e8ff47;
  padding: 10px 20px;
  border-radius: 8px;
}

📌 Fixed! (kaymaz)

✨ Fixed Kullanım Alanları:
  • ✅ Navigasyon çubukları (navbar) — her zaman üstte kalır
  • ✅ Chat butonu / "Yukarı çık" butonu
  • ✅ Bildirim toast'ları (sağ alt köşede sabit)
  • ✅ Reklam banner'ları (sayfa kaydırılsa da görünür)
📌 Fixed Özellikleri:
  • 🎯 Referans noktası → Viewport (tarayıcı görüntü alanı)
  • 📦 Normal akıştan tamamen çıkarılır (diğer elementler onun yerine kayar)
  • 🔄 Scroll yapınca sabit kalır — hareket etmez!

📌 position: absolute — Kapsayıcı Elemente Göre Mutlak

position: absolute ile bir element, en yakın positioned (relative/absolute/fixed) ebeveynine göre konumlanır. Eğer böyle bir ebeveyn yoksa, viewport'a göre konumlanır.

.absolute-box {
  position: absolute;
  bottom: 20px;
  right: 20px;
}

📌 position: relative — Normal Konumundan Kaydırma

position: relative ile bir element, normalde olacağı yerden kaydırılır. Element normal akışta kendi yerini korur (boşluk bırakır), sadece görsel olarak taşınır.

.relative-box {
  position: relative;
  top: 20px;      /* Normal konumundan 20px aşağı */
  left: 30px;     /* Normal konumundan 30px sağa */
}

📌 Relative'in En Önemli Kullanımı:

Relative, genellikle absolute elementler için referans noktası oluşturmak amacıyla kullanılır.

.parent {
  position: relative;  /* Absolute çocuklar için referans */
}

.child {
  position: absolute;  /* Parent'a göre konumlanır */
  top: 0;
  right: 0;
}

🎨 Overlay — Popüler Kullanım Alanı

Absolute + Relative kullanarak, bir resim veya kart üzerine hover'da görünen overlay (üst katman) yapabilirsiniz.

.image-container {
  position: relative;   /* Referans noktası */
  display: inline-block;
}

.overlay {
  position: absolute;   /* Container'a göre konumlanır */
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  background: rgba(0,0,0,0.7);
  opacity: 0;
  transition: opacity 0.3s;
}

.image-container:hover .overlay {
  opacity: 1;  /* Hover'da görünür */
}

📚 z-index — Katman Sıralaması

z-index, üst üste binen positioned elementlerin hangisinin önde olacağını belirler. Büyük değer önde gösterilir.

.box1 {
  position: absolute;
  z-index: 1;  /* Düşük → arkada */
}

.box2 {
  position: absolute;
  z-index: 2;  /* Yüksek → önde */
}
z-index:1

⬆️ z-index değeri büyük olan önde! (3 > 2 > 1)

⚠️ Önemli: z-index sadece positioned elementlerde (position: relative/absolute/fixed) çalışır! Static elementlerde çalışmaz.

📋 Position Özeti

  • 📍 static (varsayılan) → Normal akış, top/left çalışmaz
  • 📌 fixed → Viewport'a göre sabit, scroll'da hareket etmez
  • 🎯 absolute → En yakın positioned ebeveyne göre (yoksa viewport)
  • 📏 relative → Normal konumundan kayar, orijinal yeri boş kalır
  • 📚 z-index → Katman sırası (büyük değer önde) — sadece positioned elementlerde çalışır
  • 🔧 Absolute + Relative → Overlay, badge, tooltip, modal için mükemmel ikili!
  • ⚠️ Fixed kullanırken padding-top ile içeriği kaydırmayı unutmayın!
